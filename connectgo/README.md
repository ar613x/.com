# VERY EARLY BETA
## CONTROLS:
### Strategy
Single Click = Black Tile  
Right Click = White Tile  
Single Click on *occupied cell* = Remove Tile  
End Goal = 5-in-row (hori, vert, or diag)  
### Gameplay
Single Click = Black / White Tile (alternate)  
End Goal = 5-in-row (hori, vert, or diag)  
See rules page for more info
